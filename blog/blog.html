What are the differences between local storage and session storage?

Both are inbuild chrome storage system but these have some differences. The main difference is that local storage is permanent and session storage is temporary. It means when the data is stored in local storage it remains until the chrome browser is not uninstalled.  On the other hand, data of session storage is removed when a tab is closed. 

What are the differences between local-global scope and block scope?

Local scope and global scope mean the position of a variable. Here, if a variable is accessible from anywhere in a code is known as a global scope. On the other hand, if a variable cannot be accessed outside a block is known as block scope in Javascript.
